Projects 1: Predicting the Sale Price of Bulldozers

Objectives:

  • To predict the future sale price of bulldozers (large construction machines), given its characteristics and previous samples
  • To build a model using a random forest regressor in Colab
  • To train a model with time series data imported from CSV files
  • To evaluate a model with root mean squared log error (RMSLE) metric

Steps:

  1. Data pre-processing
    • Import data and parsing dates
    • Sort dataframe by saledate
    • Add datetime parameters for saledate column
    • Convert string to categories
    • Fill missing values
    • Turn categorical variables into numbers
    • Split data into train/validation sets
  2. Building a model
    • Build a machine learning model using random forest regressor
  3. Training a model
    • Train a model using trained dataset
  4. Model Evaluation
    • Evaluate a model with RMSLE metric
  5. Tuning hyperparameters
    • Tune hyerparameters with RandomizedSearchCV
    • Train a model with the best hyperparamters
  6. Deployment
    • Make predictions on test data
    • Extract Feature Importance

Outcomes:

  • Random forest regressor model with best hyperparameters
  • Comparison of mean RSMLE score among three models
  • Prediction on test data
  • Prediction on a custom sample
  • Comparison of feature importance



Projects 2: Multi-class Dog Breed Classification

Objectives:

  • To identify the dog breed given a dog image using a multi-class image classifier
  • To build a model using deep learning and transfer learning with TensorFlow and TensorFlow Hub
  • To train a model with 10000+ images of 120 dog breeds (size of 224x224) in Colab
  • To evaluate a model with prediction probabilities

Steps:

  1. Data pre-processing
    • Get data ready: images and labels
    • Create our own validation set
    • Preprocess images: turning into Tensors
    • Turn our data into batches
  2. Building a model
    • Build a model using Keras sequential model and MobileNetV2
  3. Training a model
    • Train a model using trained dataset
  4. Model Evaluation
    • Make predictions on validation data using a trained model
  5. Training a big dog model with full data
  6. Deployment
    • Make predictions on test dataset
    • Make predictions on custom images

Outcomes:

  • To identify the breed of a dog given an image of a dog
  • To build a multi-class image classifier using TensorFlow 2 and TensorFlow Hub
